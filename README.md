# Weather-Report

## Prepare your Linux if you don't have `npm`, `bower`, `grunt`
```
sudo apt-get install npm -g 
npm install bower -g
npm install grunt-cli -g
npm install grunt -g
```  
## Install env
```  
npm install
bower install
```
  
## Compile the project
```
grunt
```
## Run the project
```
python -m SimpleHTTPServer
```
